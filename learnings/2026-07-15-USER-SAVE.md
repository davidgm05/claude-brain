# 2. ELIMINAR ERRORES SILENCIOSOS

Actualmente existen bloques que ocultan errores:

```js
try {
  fn(t);
} catch (e) {}
```

Esto está prohibido.

Sustituye todos los `catch` vacíos por:

```js
catch (error) {
  console.error('[MODULE_NAME]', error);
}
```

Durante desarrollo añade:

```js
window.addEventListener('error', event => {
  console.error('[GLOBAL ERROR]', event.error || event.message);
});

window.addEventListener('unhandledrejection', event => {
  console.error('[UNHANDLED REJECTION]', event.reason);
});
```

La versión final debe tener cero errores de consola durante una partida completa.

# 9. DRAG AND DROP DE CALIDAD

Mantén la base actual de Pointer Events, pero pule la sensación.

## Al recoger

* Escala entre `1.025` y `1.045`.
* Elevación visual.
* Sombra separada.
* Sonido breve.
* Vibración opcional.
* Respuesta inmediata.

No debe existir retraso perceptible.

## Durante el movimiento

Usar:

```css
transform:
  translate3d(var(--drag-x), var(--drag-y), 0)
  rotate(var(--drag-rotate))
  scale(var(--drag-scale));
```

Actualizar solo mediante `requestAnimationFrame`.

Calcular inclinación según velocidad:

```js
const velocityX = currentX - previousX;
const maxTilt = packageMass > 1.25 ? 1.5 : 3;
const tilt = clamp(velocityX * 0.08, -maxTilt, maxTilt);
```

Aplicar suavizado:

```js
visualTilt += (tilt - visualTilt) * 0.2;
```

## Sombra

La sombra debe:

* Permanecer cerca de la cinta.
* Reducirse ligeramente al elevar.
* Aumentar blur.
* Desplazarse en dirección opuesta.
* Volver suavemente al soltar.

## Límites

El paquete nunca debe desaparecer completamente fuera del viewport.

Mantener al menos un 35 % visible.

## Magnetismo

Al acercarse a un destino:

* Atracción suave.
* El contenedor abre su mecanismo.
* El paquete se orienta.
* No clasificar hasta soltar.
* No revelar si el destino es correcto.

## Soltar fuera

* Regreso con resorte.
* Duración según masa.
* Pequeño impacto al volver.
* Sin penalización.
* Sin teletransporte.

## Soltar dentro

* Mover al centro del destino.
* Adaptar rotación.
* Reducir escala.
* Entrar detrás del frontal.
* Ejecutar reacción del contenedor.
* Cargar siguiente paquete durante el final de animación.
