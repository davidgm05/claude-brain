# "Context-Aware Skill Injection expanded with scoring algorithm and trigger maps"

**Subtitle:** "Agency Brain spec now includes multi-dimensional analysis, conflict resolution protocol, and complete trigger maps by domain"


## Narrative
"The Agency Brain skill injection system was significantly expanded. Previously just a keyword table, it now includes a full scoring algorithm with base scores, context boosts (1.5x for inkcalendar-agency project), recency boosts (1.2x for recently used skills), and diversity bonuses. The spec now includes complete trigger maps for Core System skills (daemon, redis, lead-scout), Marketing skills (SEO, social media, copywriting), Technical skills (React, Node, database), Business skills (WhatsApp, ecommerce), and Productivity skills (automation, content). A conflict resolution protocol handles skill disagreements, with safety-guidance.md having final say on security decisions."
