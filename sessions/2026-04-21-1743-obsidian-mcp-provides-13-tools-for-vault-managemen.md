# "obsidian-mcp provides 13 tools for vault management"

**Subtitle:** "Discovered comprehensive set of MCP tools for read/write/search/tag operations on Obsidian vaults."


## Narrative
"Discovered the full capabilities of the obsidian-mcp package. It provides a comprehensive set of 13 tools for interacting with Obsidian vaults including full CRUD operations on notes, directory management, vault-wide search, and tag management. The tool design enforces good practices by requiring separate filename and folder parameters rather than path strings. This gives the foundation needed to build a memory system - notes can be created, searched, tagged, and organized programmatically during Claude sessions."


## Facts
"[""obsidian-mcp v1.0.6 by StevenStavrakis provides 13 MCP tools"",""Tool names: read-note, create-note, edit-note, delete-note, move-note, create-directory, search-vault, add-tags, remove-tags, rename-tag, manage-tags, list-available-vaults"",""Tools support folder-based organization (filename must not contain path separators)"",""Tag management capabilities: add, remove, rename tags across notes"",""Multi-vault support via list-available-vaults tool"",""Requires Node.js 20+ and vault directory read/write access""]"
